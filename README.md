## Install

```bash
npm install muse-ui -save
```

or

```
yarn add muse-ui
```

## Get Started

```javascript
import Vue from 'vue'
import MuseUI from 'muse-ui'
import 'muse-ui/dist/muse-ui.css'
Vue.use(MuseUI)
```
